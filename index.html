Guide Make — 8 étapes
  1. Créer un nouveau scénario dans Make
  2. Trigger : Telegram Bot → Watch Messages
  3. Créer un bot via @BotFather sur Telegram et coller le token
  4. Ajouter un module HTTP → Make a request
  5. URL : VOTRE-URL/api/telegram — Méthode POST
  6. Body (JSON) : {"message": "{{1.message.text}}"}
  7. Header : x-api-token: VOTRE-TOKEN
  8. Réponse → Telegram → Send a Message avec le body de réponse
